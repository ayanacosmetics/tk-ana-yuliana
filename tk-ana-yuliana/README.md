# TK. Ana Yuliana

Gratis: GitHub + Vercel + Google Sheets + Apps Script.

## Cara pakai
1. Upload folder ini ke GitHub.
2. Deploy repo ke Vercel.
3. Buka Google Sheets > Extensions > Apps Script.
4. Paste isi gas/Code.gs.
5. Deploy > New deployment > Web app.
6. Execute as: Me. Access: Anyone.
7. Copy URL Web App.
8. Paste ke js/config.js.
9. Commit ulang. Vercel auto update.
